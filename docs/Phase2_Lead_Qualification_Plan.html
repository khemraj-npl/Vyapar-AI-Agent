8. Sales Mode — Unavailable Package

Triggers

  • detect_buying_intent(text) == True
  • sales/pricing intent + speed not in catalog
  • search_products() empty but speed/buying mentioned
  • Requested speed higher than max catalog product

Behavior when no exact match

Do NOTDO
Dump full package list in promptAcknowledge customer need first
Repeat FAQ-style catalogPropose one closest alternative
Ignore lead contextAsk for phone if missing
—Create/update lead in lead_manager
