5. Root Causes (Architecture)

  1. No turn router (intro / info / sales / repair / escalate)
  2. No conversation state (phone_collected, pitch_sent, phase)
  3. No response validator (block repeat pitch, block phone re-ask)
  4. Single-turn LLM + long prompt → template loop
  5. No Telegram inline keyboards for package selection

Phase 2 deploy improved logs; this test shows UX still fails.
